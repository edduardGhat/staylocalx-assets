# Staylocalx — Public Assets CDN

This repo serves as a free public CDN for Staylocalx Instagram content.

## Structure

```
launch/
├── master_en/   → @thestaylocalx + @staylocalx.app (€500/mo)
├── us/          → @staylocalx.us ($500/mo)
├── es/          → @staylocalx.es (€500/mes)
└── mx/          → @staylocalx.mx ($10,000 MXN/mes)

tutorials/      (future)
ads/            (future)
success_stories/ (future)
```

## URL pattern

Raw images are served at:
`https://raw.githubusercontent.com/edduardGhat/staylocalx-assets/main/<path>`
